# Reporting Security Issues

We take security bugs in `skillful-alhazen` seriously.
We appreciate your efforts to responsibly disclose your findings,
and will make every effort to acknowledge your contributions.

To report a security issue, please open a GitHub issue or email gullyburns@gmail.com
with the word "SECURITY" in the subject line.

Skillful-Alhazen is intended as research-level code enabling agent-based investigation
of knowledge from multiple domains in the wider community.
